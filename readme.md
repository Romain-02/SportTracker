# Sport Tracker

## Sessions / Entrainements

- Créer des entrainements et les sessions associées
- Synchroniser ses séances avec Strava
- Visualiser les prochaines séances

## Dashboard (statistiques)

- Voir des statistiques complètes avec des graphiques, des totaux de distance et moyenne de vitesse et nombre de séances sur les dernières dates et records en fonction du sport, d'une distance d'un temps

## Programmes

- Pouvoir créer des programmes
- Associer des entrainements au programme
- Définir des objectifs

## Lancement de l'application

````
npm install
ionic serve
````

## Déployer l'application

````
ionic build
npx cap sync
npx cap open android
````
